개발 목표 -> 학교 형광등을 esp32를 이용하여 깔끔한 앱을통해 제어

방법1: 릴레이 혹은 모스펫(or트렌지스터)을 이용하여 학교선을 직접 납떔하여 제작.
    장점1: 높은 정확성
    장점2: 형광등의 밝기를 조절할 수 있을 수도 있음
    장점3: 스위치의 외형에 큰 영향이 없음

    단점1: 학교 기물에 마음대로 손댈 시 문제가 될 수 있음
    단점2: 높은 전류나 전압으로 인해 문제가 생길 수 있음
    단점3: 내가 학교의 전기 시스템에 대하여 정확히 알고 있지 않음

방법2: 서보 모터 혹은 모터를 이용하여 직접적으로 스위치를 on off하는 시스템
    장점1: 학교 기물에 파손 가능성이 낮음
    장점2: 수리가 용이함
    장점3: 높은 전류나 전압을 다루지 않음

    단점1: 만들기가 구조적으로 좀 더 복잡함
    단점2: 스위치 외형에 영향이 큼
    단점3: 여러 스위치와 호환이 조금 어려움

내가 진행하려는 프로젝트 자체가 학교에서 진행하는것 이므로 용이성이나 편의성 보다는 안정성이 더 중요함
